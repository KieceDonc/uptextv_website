Features

Create groups 

This feature allow to create groups and add any streamers you want

Hide / show groups

Allows you to temporarily or permanently hide or show a group

Sort streamers

Sort your pinned streamers with custom parameters: game name, streamer name, viewer, uptime ... 

Reorder groups

Reorder groups to get your favorites groups on top
